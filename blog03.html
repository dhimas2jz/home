Pengembangan Game 2D

Rearrange sebagai sebuah tim developer baru sedang mengembangkan game 2D dan game 3D, dimulai dengan game 2D simpel, yang nantinya akan lanjut ke game dengan tingkat kesulitan medium dan hard, sebagai developer baru tentunya harus memulai dari kecil terlebih dahulu.

Diharapkan dengan perkembangan yang dimiliki tim developer ini, bisa membuat manfaat untuk semuanya, karena pasti akan ada perkembangan yang signifikan dari tim developer ini, tidak hanya membuat game 2D dan game 3D namun kedepannya diharapkan bisa membuat aplikasi dan layanan lainnya juga.
